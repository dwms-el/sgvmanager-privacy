Política de Privacidade

SGV Manager — Sistema de Gestão de Vídeo

Última atualização: junho de 2025

1. Coleta de Dados

O SGV Manager não coleta, transmite nem armazena dados pessoais em servidores externos. Toda informação gerada pelo uso do aplicativo é armazenada exclusivamente no dispositivo local do usuário.

2. Dados Armazenados Localmente

  • Configurações de monitores e playlists de conteúdo
  • Logs de operação do sistema (sem informações pessoais identificáveis)
  • Preferências de idioma e tema visual
  • Credenciais de acesso ao painel administrativo (armazenadas de forma protegida)

Esses dados ficam na pasta de dados do aplicativo no Windows e são removidos ao desinstalar o app.

3. Acesso à Internet

O SGV Manager conecta-se à internet apenas nos seguintes cenários:

  • Conteúdo Web: quando o usuário configura URLs (YouTube, dashboards, etc.) para exibição nas telas — a conexão é feita diretamente para o endereço configurado pelo próprio usuário.
  • Acesso Remoto: quando habilitado, o app abre uma porta local para gerenciamento na rede interna — sem comunicação com servidores externos.

Nenhuma informação sobre o uso do aplicativo é enviada ao desenvolvedor ou a terceiros.

4. Componentes de Terceiros

  • LibVLC / VideoLAN: biblioteca de reprodução de vídeo. Não coleta dados. Licença LGPL.
  • Microsoft WebView2: componente de navegação web. Sujeito à política de privacidade da Microsoft para o runtime WebView2.

5. Público-alvo

O SGV Manager é um aplicativo profissional voltado a empresas e não é direcionado a menores de 18 anos.

6. Contato

Para dúvidas sobre esta política de privacidade, entre em contato pelo e-mail da DW Moreira Studio disponível na página do aplicativo na Microsoft Store.
